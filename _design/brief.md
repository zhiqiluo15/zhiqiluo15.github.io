# 灵弦华夏 网站设计简要

## 1. 概述与目标
- **页面类型**: 静态首页/作品展示站
- **目标用户**: 文学/音乐爱好者
- **核心定位**: 古风文艺风格个人网站，展示作品与设定

## 2. 视觉方向
- **美学风格**: 极简古风、文艺质感、星空氛围
- **背景**: 星空动画背景，全屏铺满，自适应，不影响文字阅读，固定全局外壳容器（提升体验感）
- **整体氛围**: 神秘、优雅、有文化底蕴

## 3. 布局结构
```
┌─────────────────────────────────────────┐
│ 灵弦华夏        [导航: 作品|设定|暂定]   │
│                                         │
│         ┌─────────────┐                 │
│         │  原生播放器  │ (240px宽)      │
│         │                             │
│         └─────────────┘                 │
│                                         │
│              [主视觉区]                  │
│            标题 + 标语                   │
│                                         │
│                                         │
│            贪文笔著 © 2026-05-31        │
└─────────────────────────────────────────┘
```

## 4. 字体与排版
- **古风字体**: 左上角「灵弦华夏」使用古风字体（如 ZCOOL XiaoWei、Noto Serif SC）
- **正文字体**: 简洁无衬线字体
- **文字方向**: 从上到下，从左到右

## 5. 播放器规格
- **原生音频**: HTML5 `<audio>`，宽度 240px，默认样式


## 6. 响应式要求
- 桌面端：导航在右侧，整体居中布局
- 移动端：播放器堆叠，导航简化，文字不被遮挡，导航左右分布

## 7. 输出
- 文件: `index.html`
- 自包含 CSS（内联或 <style>）
